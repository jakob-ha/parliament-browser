# parliament-browser

## Funktionalitet

### index

En välkomstsida har en knapp som tar användaren till listan med ledamöter

### personlist

En lista med sittande ledamöten presenteras med foto (för de som har), namn och födelsedatum.

Listan kan filtreras efter text input och den sorteras efter namn.  Stigande/avtagande sortering kan togglas.

Trycker man på en ledamot tas man till deras infosida.

### personinfo

Infosidan innehåller namn, epost och ledamotens aktiva roller i olika grupper.
